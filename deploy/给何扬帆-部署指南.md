# 🚀 审核系统 - 一键部署指南（何扬帆专用）

## 第一步：装 Node.js（只需要做一次）

1. 打开 https://nodejs.org
2. 点左边绿色大按钮 **「LTS」** 下载
3. 下载完双击安装，一路点 **Next** 直到完成
4. 重启电脑（确保环境变量生效）

---

## 第二步：下载项目（只需要做一次）

**方式一：直接下载 ZIP（推荐，最省事）**

1. 打开 https://github.com/yuwangliuyi/ai-image-review
2. 点绿色 **「<> Code」** 按钮 → 点 **「Download ZIP」**
3. 把下载的 `ai-image-review-main.zip` 解压到桌面
4. 把解压出来的文件夹重命名为 `ai-image-review`

**方式二：用 Git 克隆（如果电脑已装 Git）**

1. 按 `Win + R`，输入 `cmd`，回车（打开命令提示符）
2. 依次输入以下两条命令（一条一条执行）：

```
cd /d %USERPROFILE%\Desktop
```

```
git clone https://github.com/yuwangliuyi/ai-image-review.git
```

等它跑完，桌面上会出现 `ai-image-review` 文件夹。

---

## 第三步：一键安装（只需要做一次）

1. 打开桌面上的 `ai-image-review` 文件夹
2. 进入 `deploy` 文件夹
3. **双击 `install.bat`**
4. 黑窗口会自动跳到项目根目录，然后开始安装
5. 等待自动跑完（大约 2-5 分钟），出现「安装完成」即可关闭

---

## 第四步：启动系统（每次开机后）

1. 打开 `ai-image-review` → `deploy` 文件夹
2. **双击 `start.bat`**
3. 黑窗口保持打开不要关
4. 首次会有 1-2 分钟的构建过程，看到「✓ Ready」就好
5. 浏览器打开 → **http://localhost:3002**

> ⚠ 如果打不开，双击 `deploy` 里的 **`diagnose.bat`**，把黑窗口里的内容截图发给鱼王。
