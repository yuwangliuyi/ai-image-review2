AI 图片审核系统

功能使用说明书 · 版本 1.0 · 适用于全部角色用户

1. 系统概述

AI 图片审核系统是一套面向跨境电商视觉团队的图片上传·审核·归档全流程管理平台。它覆盖了从图片上传到最终归档的完整链路，并为不同角色的用户提供差异化的数据视图和操作权限。

2. 角色与权限体系

系统内置三种角色，不同角色看到的功能和可执行的操作不同。

角色标识权限范围
管理员 (ADMIN) 系统最高权限 所有功能 + 用户管理 + 数据导出 + 创建新账号
审核员 (REVIEWER) 审核 + 查看 审核中心 · 数据中心 · 归档 · 仪表盘 · 通知
（何扬帆额外拥有数据导出权限）
上传者 (UPLOADER) 上传 + 查看 图片上传 · 数据中心（仅本人数据）· 归档（仅本人通过的 SPU）· 仪表盘 · 通知

🔐 数据隔离规则

上传者只能看到自己上传的图片和 SPU；审核员可以看到被分配给自己审核的图片；管理员可以查看全量数据。数据中心和归档列表均按此规则过滤。

3. 注册与登录

3.1 登录

进入系统首页自动跳转登录页。输入用户名 + 密码即可登录。已登录用户访问登录页会自动跳转到仪表盘。

3.2 注册新用户

⚠️ 权限限制

注册页面仅对管理员 (ADMIN)和审核员何扬帆开放。其他用户无法访问注册页，这将确保账号创建受控。

注册需要填写：用户名、邮箱、部门、密码、角色。

系统会检测用户名是否已存在，冲突时给出提示。

4. 仪表盘

登录后的默认首页，以图表形式展示系统数据概览。支持三个维度切换：

  1. 总体概览 — 全公司 SPU 数量、图片统计、通过率、驳回原因分布
  2. 部门视图 — 按部门拆分，对比各团队的上传量和通过率
  3. 个人视图 — 仅显示当前登录用户自己的数据

📊 看板内容

顶部卡片：SPU 总数 · 图片总数 · 通过率 · 归档数
图表区域：柱状图（品类分布）+ 饼图（审核状态 / 驳回原因）
自动刷新：每 30 秒自动更新一次数据

5. 图片上传

上传流程采用SPU → 图片两级结构，先创建任务再往里面塞图。

5.1 创建 SPU 任务

  1. 点击「上传图片」进入 SPU 任务列表
  2. 点击右上角 + 新建 SPU
  3. 填写 SPU 名称、选择品类、填写国家风格、选择店铺
  4. 点击「创建」——SPU 创建后进入待上传状态

💡 自动补全

品类和店铺字段支持输入即搜——输入关键词后系统自动匹配已有选项，也支持输入全新的品类或店铺名称。

5.2 上传图片

  1. 在 SPU 列表中点击目标 SPU，进入详情页
  2. 拖拽图片文件到上传区域，或点击区域选择文件
  3. 支持一次上传多张，每张图片会显示缩略图预览
  4. 可点击 ❌ 删除单张图片
  5. 上传后图片状态为 待审核

📁 文件规范

支持 JPG / PNG / WebP 格式，单文件建议不超过 20MB。
系统会自动按「品类 / 店铺 / SPU 名称 / 国家风格」生成结构化存储路径。

6. 审核中心

审核员在此逐张审核图片，是系统的核心工作台。

6.1 审核界面

  • 左侧：大图预览，点击可灯箱放大查看细节
  • 右侧：图片信息（文件名、大小、所属 SPU、品类、上传者）
  • 底部缩略图条：可切换当前 SPU 下的其他图片
  • 顶部筛选栏：按审核状态筛选（待审 / 已通过 / 已驳回）

6.2 审核操作

操作快捷键效果
✅ 通过Enter 或 A图片状态变为「已通过」，自动跳至下一张
❌ 驳回R弹出驳回意见框，填写原因后提交，自动跳至下一张
⏭ 跳过S 或 →保留当前状态，跳至下一张
⬅ 上一张←回到上一张图片
🔍 放大Space灯箱模式全屏查看

📝 审核记录

每张图片的右侧面板会显示完整审核历史，包含审核人、操作、时间、驳回意见。即使图片被重复审核，历史记录也会完整保留。

6.3 SPU 归档触发

当某个 SPU 下所有图片都通过后，该 SPU 自动进入归档列表，无需手动操作。

7. 数据中心

以表格形式展示系统所有明细数据，支持查询、筛选和翻页。

三个标签页

标签展示内容
📦 SPU 任务SPU 名称、品类、国家风格、店铺、状态、上传者、审核员、创建时间、图片统计
🖼 图片列表文件名、大小、所属 SPU、品类、上传者、批次号、审核状态、上传时间
📋 审核记录图片名、审核人、操作 (通过/驳回)、驳回意见、审核时间

筛选与搜索

支持按审核状态（全部 / 待审 / 通过 / 驳回）筛选；
支持关键词搜索 SPU 名称或文件名；
每页 20 条记录，底部有分页导航。

8. 归档管理

审核全部通过后的 SPU 会自动出现在「归档管理」页面。该页面提供：

  • 归档列表：SPU 名称、品类、国家风格、店铺、图片数量、归档时间
  • 🔽 下载按钮：点击将整个 SPU 的图片打包下载为 ZIP
  • 分页浏览：支持翻页查看历史归档

📥 下载说明

每个 SPU 独立打包下载。文件按「品类/店铺/SPU名称/国家风格/文件名」的目录结构组织在 ZIP 内。
下载按钮仅对通过审核的上传者和管理员可见。

9. 站内通知

系统在以下场景自动发送通知给上传者：

  • SPU 审核全部通过 ✅
  • 图片被驳回（含驳回原因）❌

通知入口

顶部导航栏右侧的 🔔 铃铛图标 显示未读通知数量（红点角标）。
点击铃铛进入通知列表页，可查看所有通知详情。
支持 「全部已读」 一键标记。

10. 用户管理

🔒 仅管理员可访问

「用户管理」入口仅对 ADMIN 角色可见（左侧导航栏底部「系统管理」区域）。

功能

  • 查看所有用户列表：姓名、部门、角色、邮箱、注册时间
  • 按角色筛选（全部 / 管理员 / 审核员 / 上传者）
  • ➕ 创建新用户（跳转至注册页）
  • 🗑 删除用户（弹出二次确认）

11. 数据导出

🔒 权限

仅管理员 (ADMIN) 和审核员 何扬帆 可见导出按钮。

在「数据中心」页面右上角，点击绿色 「导出全部数据」 按钮，系统将生成一个 Excel 文件（.xlsx），包含 4 个工作表：

Sheet内容
用户汇总姓名、部门、角色、邮箱、注册时间、上传 SPU 数、上传图片张数、通过/驳回/待审图片数、审核记录数
SPU 明细SPU 名称、品类、国家风格、店铺、状态、上传者/部门、审核员、创建时间、图片总数及各状态数量
图片明细文件名、大小、类型、所属 SPU、品类、上传者/部门、批次号、审核状态、审核人/部门、审核意见、时间
归档明细SPU 名称、品类、国家风格、店铺、上传者、部门、图片数量、归档时间
